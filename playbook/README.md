# Syntra V2 Strategy
## 策略
自适应市场机制策略 - 在趋势市场中跟随趋势，在震荡市场中均值回归，在市场不明朗时保持观望。
## 开仓规则
- **趋势市场**: 当SMA 20 > SMA 50时买入，当SMA 20 < SMA 50时卖出
- **震荡市场**: RSI < 30时买入（超卖），RSI > 70时卖出（超买）
- **不明朗市场**: 保持观望
## 平仓规则
- 止盈: +4%
- 止损: -2%
- 最大持仓时间: 4小时
## 风险控制
- 最大仓位: 1 BTC
- 最大回撤: 30%
- 每日交易限制: 10笔
## 策略优势
- 自动适应市场变化
- 结合趋势跟踪和均值回归
- 明确的风险管理规则
- 在不明朗市场中保持观望
